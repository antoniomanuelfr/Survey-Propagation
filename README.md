# Survey Propagation
This project is a C++ implementation of the Survey Propagation algorithm, proposed on this paper:
 https://arxiv.org/pdf/cs/0212002.pdf

## Structure
1. `cnf`: Folder that storage DIMACS files.
2. `inc`: Headers.
3. `src`: Source code.
4. `doc`: Doxygen generator file.
## Usage
1. `mkdir build`
2. `cd build && cmake ..`
3. `make`
